# Framework boundaries

This document maps common AI-agent ecosystem primitives to this framework's portable layer.

The framework boundary is the bootstrap chain, command routers, private route playbooks, role metadata, repo-local helpers, validation checks, and collab state protocol. This document is a boundary map, not a capability registry, runtime contract, or host integration specification.

The runtime path remains the bootstrap anchor. Codex and Claude connect through the bootstrap chain; host-native features stay outside the portable layer unless a route explicitly documents a harness-specific policy.

MCPs are out of scope for the portable layer by design. They may be useful host integrations, but they are not required to explain or operate this framework.
