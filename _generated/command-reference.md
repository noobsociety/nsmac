# Command Reference

**Audience:** A non-Cursor agent looking up slash commands, flags, and valid values without Cursor autocomplete. This document is the authoritative reference for routing-only prose dispatch forms and argument contracts.

## Value Classes

Every argument and flag in this reference is classified as one of three value classes:

- **`literal`** — the value is a fixed set known at render time; every valid member is listed. No values outside the listed set are accepted.
- **`type`** — the value is free-form or open-ended; the reference names the type and any constraint (e.g., `free text`, `markdown path`, `collab slug`).
- **`dynamic`** — the valid values come from live state and cannot be enumerated statically; the reference names the source and the resolution rule (e.g., `collab slug, id, or #N; defaults to active collab`).

## Entry Layout

Each command entry begins with the **routing-only prose dispatch form** — the literal `(namespace command arg ...)` reference form. It is not terminal-executable; resolve it through the matching slash route instead of copying it into a shell. This form is listed for every route without exception; no route omits it on the grounds that it resembles the slash form.

Below the prose dispatch form, each argument or flag occupies one indented line:

```
(namespace command --flag <value>)
  <name>    required|optional    <class>: <detail>
```

Namespace sections group related commands under a `## <namespace>` heading.
