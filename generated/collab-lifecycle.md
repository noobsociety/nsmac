## Phase Table

| Phase | Speak rule | Auto-advance exempt | Moderator excluded | Reviewer rule |
| --- | --- | --- | --- | --- |
| Audit | one contribution per required role | no | no | required last in convergent reviewer mode |
| Discussion | open discussion turns | yes | no | optional |
| Conclusion | one contribution per required role | no | yes | required last in convergent reviewer mode |
| Action Plan | one contribution per required role | no | yes | none |
| Handoff | one contribution per required role | no | yes | none |
| Completion | execution records, then reviewer seal when configured | yes | yes | none |

| Field | Meaning |

| `moderatorRole` | Participant that starts the collab and is removed from later implementation phases. |

| `verification.rounds` | Registry-counted reviewer/executor paired events in `Completion.verification`. |
| `verificationSeal` | Reviewer seal object that binds observed registry revision, execution entries, validation scopes, and touched paths. |
| `activeCollabId` | Registry pointer for the selected open collaboration record. |
